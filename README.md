# Судебный календарь 
Этот веб-приложение представляет собой удобный инструмент для отслеживания расписания судебных заседаний в арбитражных судах.
Позволяет просматривать расписание сразу нескольких кабинетов в одном окне, а также открывать несколько сайтов одновременно для удобства сравнения информации.

## Основные возможности:
- Просмотр расписания различных кабинетов в едином интерфейсе.
- Возможность открытия нескольких сайтов с расписаниями в одном браузере.
- Открытие расписания судей,которые сегодня рассматривают дела.

## Основной stack
- Python 3.10
- Django 5.1.2

## Установка и запуск:
Для установки и запуска приложения выполните следующие шаги.
1. Клонируйте репозиторий.
2. Установите необходимые зависимости - requirements.txt
3. Выполните миграцию базы данных.
4. Запустите сервер разработки.
